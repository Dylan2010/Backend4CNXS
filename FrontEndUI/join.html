  • CTO技术总监
    岗位职责：
    1. 负责大数据平台技术框架的选型和开发、整体架构和引擎的规划和设计，软件和系统性能预测，性能规划和性能优化
    2. 通过大数据处理对数据进行分析、建模、挖掘，开发，对应用开发团队提供技术指导、方案规划及计算模型设计、项目审核的评估
    3. 负责和互联网业界其他大数据公司的横向合作和沟通
    4. 负责技术团队建设和管理，注重大数据核心技术团队的人才培养和营造创新的研发文化
    5. 组织研究行业最新产品的技术发展方向，主持制定技术发展战略规划

    任职条件：
    1. 博士学历，计算机科学与技术相关专业优先，七年以上软件开发经验，其中包括两年以上互联网、电商软件开发工作经验，五年以上团队管理经验
    2. 精通PHP、Java、Oracle，MySQL，DB2，熟悉Linux系统和Mysql数据库的管理和开发，具备大数据架构管理经验
    3. 熟练应用 Hadoop 大数据生态工具（至少包含 Hive、Spark 二次发开能力）
    4. 具备全面的软件开发、系统管理、网络安全等方面的技术能力
    5. 互联网平台的架构设计、系统分析、软件实现、性能优化、系统安全管理等
    6. 熟悉大型网站架构、数据库、系统运维以及常用数据挖掘算法，如逻辑回归、决策树、关联规则、序列模式、时间序列、SVM、贝叶斯、聚类。
    7.工作积极主动，具备很强的责任心、团队合作精神、良好的语言表达及沟通交流能力。
  • 大数据工程师
    岗位职责：
    1、负责搭建并维护公司大数据系统：Hadoop集群及HBase/Hive集群, 保证其高可用性和稳定性；
    2、负责生产数据系统的数据接入大数据系统的规划和实施；
    3、负责设计实现可支撑公司业务增长需要的Hadoop/HBase/Hive集群运维平台与工具；
    4、负责公司大数据系统各个组件（包括Hadoop集群，HBase集群，Hive集群等）的业务监控、持续交付、应急响应、容量规划等；
    5、研究Hadoop集群运维相关技术，持续优化集群架构，发现并解决重大故障及性能瓶颈

    任职条件：
    1、硕士及以上学历，5-7年以上大数据开发经验；
    2、熟悉Hadoop、Storm、Spark、Flume、Kafka、Hbase等组件的原理，有良好的系统性能优化及故障排除能力；
    3、负责和参与大数据基础架构平台的监控、资源管理、数据流管理，能够开发各种Hadoop大数据自动化运维与监控工具；
    4、对Hadoop平台架构能够不断优化，提升数据产品的质量和响应速度；
    5、精通Shell/Python/Java语言的一种或多种；
    6、有大规模hadoop运维经验者优先，有hadoop/hbase开发经验者优先。
  • 信息技术支持
    岗位职责：
    1. 协助公司IP技术工程团队研究和制订公司产品开发的技术架构
    2. 关注技术架构的最新发展动向，积极与技术总监,产品经理进行沟通，有效提出利于公司技术架构的发展规划
    3. 编制公司技术架构指导手册并进行维护更新
    4. 按照公司要求按时保质地完成公司研发产品的系统架构设计任务
    5. 负责配合公司IP数据产品的技术架构实现并落实分解到具体架构平台上
    6. 负责在公司软件产品的研发过程中监控项目开发在架构上的准确执行
    7. 为项目开发团队提供架构上的配合和帮助

    任职条件：
    1.本科以上学历，具有5年以上担任系统架构师的经验，参与JAVA平台中大型项目2个以上经验
    2. 熟悉当前多种主流的架构平台，懂大数据技术：Hadoop、Hive、HBase、Spark、Impala 懂数据库：Oracle、DB2、MySQL、SQL Server、GreenPlum，懂PaaS/IaaS/SaaS的优劣势，对云服务平台的结构有全面和深入了解，有阿里云或Azure的应用平台搭建经验者优先
    3.具有大型互联网应用平台的系统架构设计经验，深入掌握压力处理机制，熟悉当前主流SLB产品的优劣势
    4. 精准认知IP大数据机制的核心结构，并具有数据安全机制的成功设计经验
    5. 熟悉C#/AJAX/XML/CSHTML/CSS/RESTful API等主流互联网开发技术，
    6. 熟练掌握SQL Server/MySql，熟悉MongoDB,, Oracle等至少一种数据库及查询语言，多种关系型数据库，熟悉脚本语言、数据建模或拥有海量数据处理和挖掘经验者优先；
    7. 精通MVC开发架构，熟练掌握SOA/SaaS/云计算/中间件等技术原理，有实际相关项目经验者优先
    8. 善于沟通，有极强的团队合作能力，责任心强，有攻克困难任务的决心和能力
